A high-quality plugin for OBS Studio that allows you to use your Android or iOS device as a camera source.

## 🚀 Features
- **High Resolution**: Supports 1080p and up to 4K UHD video transfer.
- **Low Latency**: Optimized for minimal delay via WiFi or USB.
- **Hardware Acceleration**: Works seamlessly with DXVA, VideoToolbox, NVIDIA (NVENC), and VAAPI.
- **Multi-Device**: Connect multiple phones as separate sources.


## 🛠 Installation and Usage

Visit [droidcam.app/obs](https://droidcam.app/obs/#top) for installation and usage.
